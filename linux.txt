cd 切换目录
   - 刚才工作目录
   ~ 用户的目录
   .表示当前目录
   ..表示上级目录
   / 查看根目录	
	
pwd 查看当前目录
ls 查看目录列表
	  -a 查看所有文件包括隐藏文件
	  -l 列表显示文件
	  -h 显示文化的信息和列表 大小等等
ifconfig 查看网络网卡信息

vim  打开文件
touch 创建文件
cp 复制文件
rm 删除文件
rm -f 不提示删除
mv 移动文件
mkdir 创建目录
rmdir 删除目录
grep 搜索
find 搜索文件
whereis  搜索程序的路径
gzip 压缩文件
yum 软件中心

Samba服务器  可和本机共享


终端编辑工具 v1/vim  emacs
软件安装: yum install -y vim
